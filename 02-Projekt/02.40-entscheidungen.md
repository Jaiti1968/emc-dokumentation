# Entscheidungen – EMC Mitgliederverwaltung

# 1. Zweck des Dokumentes

Dieses Dokument dient als langfristiges Projektgedächtnis der EMC Mitgliederverwaltung.

Es dokumentiert strategische, organisatorische, fachliche und technische Entscheidungen, die Auswirkungen auf Produkt, Projekt oder Dokumentation besitzen.

Ziel ist die langfristige Nachvollziehbarkeit wesentlicher Entscheidungen.

---

# 2. Governance für Entscheidungen

## Nummerierung

Jede Entscheidung erhält eine eindeutige Kennung.

Schema:

```text
E-001
E-002
E-003
...
```

---

## Status

Mögliche Statuswerte:

| Status    | Bedeutung                                                 |
| --------- | --------------------------------------------------------- |
| Aktiv     | Entscheidung ist gültig                                   |
| Ersetzt   | Entscheidung wurde durch eine andere Entscheidung ersetzt |
| Verworfen | Entscheidung wurde zurückgenommen                         |

---

## Aufbau einer Entscheidung

Jede Entscheidung enthält:

- Titel
- Status
- Datum
- Kontext
- Entscheidung
- Begründung
- Auswirkungen

# E-001

## Titel

EMC-INFRA ist technische Source of Truth

**Status:** Aktiv

**Datum:** 09.06.2026

### Kontext

Infrastruktur-, Betriebs-, Security-, Monitoring-, Backup- und Recovery-Dokumentation existieren projektübergreifend.

### Entscheidung

EMC-INFRA ist die technische Source of Truth für alle Infrastruktur- und Betriebsaspekte.

### Begründung

Vermeidung von Redundanzen und widersprüchlicher Dokumentation.

### Auswirkungen

- Infrastrukturdokumentation verbleibt in EMC-INFRA
- Produktdokumentation verweist auf EMC-INFRA
- technische Betriebsprozesse werden dort gepflegt

# E-015

## Titel:

Repositorystrategie Produktdokumentation

### Entscheidung:

Langfristig wird die Produkt- und Projektdokumentation der EMC Mitgliederverwaltung
in einem eigenständigen Repository emc-mitglieder-dokumentation geführt.

Bis zur Umsetzung verbleibt die Dokumentation im bestehenden Repository EMC-DOKUMENTATION.

### Begründung:

Die Dokumentation betrifft ausschließlich das Produkt emc-mitglieder.
Zukünftige Produkte (emc-finanzen, emc-noten) sollen eigene Dokumentationsstrukturen
erhalten können.

### Auswirkungen:

- langfristiges Zielrepository: emc-mitglieder-dokumentation
- kontrollierte Migration
- keine Vermischung unterschiedlicher Produktdokumentationen
