In Access kann man eine Nutzungsvereinbarung erstellen. Die Vorlage bildet dazu eine Nutzungsvereinbarung.dotm, deren Pfad aber fest im Code verdrahtet ist. Deshalb funktioniert sie nur auf den einem Rechner, wo die Vorlage auch entsprechend hinterlegt wurde. Richtig wäre, die dotm auf das NAS zu bringen und immer beim Kopieren der ACCESS-Arbeitskopie mit runterzuladen. In Access muss dann der relative Pfad stehen, wo auch die ACCESS-Datei zu finden ist.
